# slime99

## Package Contents

- slime99.exe: Graphical version of the game, rendering with vulkan. Prefer this over slime99-opengl.exe.
- slime99-opengl.exe: Alternative graphical version of the game, rendering with opengl. Try this if slime99.exe doesn't work.
